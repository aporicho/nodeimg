# 节点

> 画布中单个节点的视觉结构和外观配色。参数控件系统详见 [2.1.3](./2.1.3-param-widgets.draft.md)。

## 节点结构

节点最多分四区，content 区可选：

```
┌─────────────────────┐  ← header：节点名 + 分类配色背景
│  文本生成            │
├──────────────────────┤  ← content（可选）：节点的主体内容
│ Lorem ipsum...       │    文本节点 = 文本编辑框
│                      │    图片节点 = 图片预览
├──────────────────────┤  ← params：参数控件列表（可折叠）
│ ● 提示词  [......]   │
│ ● 模型    [GPT-4 ▼] │
│           [ 生成 ]   │
├──────────────────────┤
│ ○ 输入               │  ← 输入引脚（左侧）
│              输出 ○ │  ← 输出引脚（右侧）
└─────────────────────┘
```

- **Header**：节点名称，背景色按分类配色
- **Content**（可选）：节点内联内容区，由节点类型定义是否存在及其渲染方式。图片节点显示图片预览，文本节点显示文本编辑框，视频节点显示完整视频播放器（含播放/暂停、进度条，帧解码 + GPU 纹理渲染）
- **Params**：参数控件列表，支持折叠；引脚圆点紧靠左/右边缘
- **引脚**：圆点颜色按数据类型配色，悬停时高亮显示兼容性

## 显示状态

节点根据 `UIState.selection` 切换两种显示状态：

**未选中（折叠）**
```
┌─────────────────────┐
│  文本生成            │  ← header
├──────────────────────┤
│ Lorem ipsum...       │  ← content（若有）
├──────────────────────┤
│ ○ 输入    输出 ○    │  ← 引脚
└─────────────────────┘
```

**选中（展开）**
```
┌─────────────────────┐
│  文本生成            │  ← header
├──────────────────────┤
│ Lorem ipsum...       │  ← content（若有）
├──────────────────────┤
│ ● 提示词  [......]   │  ← params（展开）
│ ● 模型    [GPT-4 ▼] │
│ ● 温度    [--0.7--] │
│           [ 生成 ]   │
├──────────────────────┤
│ ○ 输入    输出 ○    │  ← 引脚
└─────────────────────┘
```

无 content 区的普通节点（如亮度/对比度）选中时同样展开 params，未选中时折叠为紧凑形态。

## 配色

### 分类配色（Header）

节点头部按分类用不同色调区分。分类通过 `CategoryRegistry` 注册，颜色随分类一起声明，新增分类无需修改主题文件：

```rust
category_registry.register(CategoryDef {
    id:            "data",
    label:         "输入 / 输出",
    default_color: Color::hex("#4A90D9"),
});
```

内置分类（供参考，非硬编码）：

| 分类 | 色调 | 代表节点 |
|------|------|---------|
| `data` | 蓝 | LoadImage、SaveImage |
| `generate` | 紫 | 纯色、渐变、噪声 |
| `color` | 橙 | 亮度、曲线、色调 |
| `transform` | 绿 | 缩放、裁剪、旋转 |
| `filter` | 粉 | 模糊、锐化、降噪 |
| `composite` | 金 | 混合、蒙版 |
| `tool` | 灰 | 预览、调试工具 |
| `ai` | 紫红 | 采样、模型加载 |

### 数据类型配色（引脚）

引脚圆点按数据类型着色，用户通过颜色判断连线兼容性：

| 数据类型 | 色调 |
|---------|------|
| `image` | 蓝 |
| `mask` | 青绿 |
| `float` | 橙 |
| `int` | 青 |
| `string` | 灰 |
| `color` | 紫 |
| `bool` | 红 |
| `model` | 亮紫 |
| `clip` | 金 |
| `vae` | 红 |
| `conditioning` | 淡紫 |
| `latent` | 粉 |

## 布局常量

| 常量 | 值 | 说明 |
|------|---|------|
| 节点圆角 | 6px | 节点体和头部的圆角半径 |
| 节点最小宽度 | 180px | 节点在画布上的最小宽度 |
| 引脚半径 | 5px | 输入/输出引脚圆点半径 |
| 面板内边距 | 8px | 节点内边距 |
| 控件间距 | 4px | 参数控件之间的垂直间距 |


## 执行状态显示

节点根据 `UIState.execution` 中的状态显示视觉反馈：

| 状态 | 显示 |
|------|------|
| 等待执行 | 无变化 |
| 执行中 | header 显示进度条或旋转指示器 |
| 执行完成 | 恢复正常 |
| 执行出错 | header 变红，显示错误图标 |
