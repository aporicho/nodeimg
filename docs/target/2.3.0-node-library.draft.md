# 节点库

> 所有可用节点的搜索和添加入口。弹出式浮层，顶部搜索框 + 下方分类菜单。

## 触发方式

| 触发 | 说明 |
|------|------|
| 双击画布空白区域 | 在双击位置弹出，添加的节点放置在该位置 |
| 拖出连接线后在空白处松开 | 在松开位置弹出，选择节点后自动与源引脚连接 |

选择节点或点击浮层外部时关闭。

## 布局

```
┌─────────────────────┐
│ 🔍 搜索节点...       │  ← 搜索框（自动聚焦）
├─────────────────────┤
│ ▼ 输入 / 输出        │
│     加载图像         │
│     保存图像         │
│ ▼ 颜色调整           │
│     亮度 / 对比度    │
│     曲线             │
│ ▶ 滤镜               │
│ ▶ AI 生成            │
└─────────────────────┘
```

弹出时搜索框自动聚焦，用户可以直接输入过滤。

## 搜索

- 关键词实时过滤，匹配节点名称
- 有搜索词时隐藏分类分组，平铺显示匹配结果

## 添加节点

单击节点名，节点创建在触发位置。若由连接线拖出触发，自动与源引脚连线（类型兼容时）。

添加操作发给交互控制器，由交互控制器调用引擎 `add_node` API。
