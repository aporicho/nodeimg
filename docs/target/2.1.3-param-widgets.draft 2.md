# 参数控件系统

> 节点 body 内的交互控件层。根据参数类型和约束自动映射控件，支持覆写自定义控件。

## 职责

- 接收用户输入，通过画布控制器调用 `set_param`
- `set_param(preview=true)`：拖动中实时预览，不入 undo 栈
- `set_param(preview=false)`：松手确认，入 undo 栈
- params 区始终展开显示，无折叠（变更：原为节点未选中时折叠隐藏，改为始终可见）

## 默认映射

`WidgetRegistry` 根据参数的 `DataType + Constraint` 映射控件类型：

| DataType | Constraint | 控件 |
|----------|-----------|------|
| `Float` | `Range(min, max)` | `Slider` |
| `Float` | 无 | `DragValue` |
| `Int` | `Range(min, max)` | `SliderInt` |
| `String` | `Enum(variants)` | `Dropdown` |
| `String` | `Enum`（选项 ≤ 4） | `RadioGroup` |
| `String` | `FilePath` | `FilePicker` |
| `String` | `Multiline` | `TextArea` |
| `String` | 无 | `TextInput` |
| `Bool` | — | `Checkbox` |
| `Color` | — | `ColorPicker` |
| `Image` | — | `ImagePreview` |
| — | `action` | `ActionButton` |

## 注册新控件类型

`WidgetRegistry` 支持注册全新的控件类型，不只是覆写：

```rust
widget_registry.register(WidgetEntry {
    // 匹配条件：DataType + Constraint 组合
    match_fn:  Box::new(|dt, constraint| {
        dt == DataType::Float && constraint == Some(Constraint::Curve)
    }),
    // 渲染函数：返回 iced widget
    widget_fn: Box::new(|param, value, on_change| {
        CurveEditor::new(value, on_change)
    }),
    priority: 10, // 高优先级先匹配
});
```

注册的条目按 `priority` 排序，第一个匹配的条目生效。内置映射（Slider、Dropdown 等）优先级最低，自定义条目优先级更高时会覆盖。

## 控件覆写

节点定义时可显式指定控件，覆盖默认映射：

```rust
node! {
    name: "ColorGrade",
    params: [
        param!("curve",    DataType::Float, widget: CurveEditor),
        param!("strength", DataType::Float, Constraint::Range(0.0, 1.0)),
    ],
}
```

节点文件夹内存在 `widget.rs` 时，框架跳过 `WidgetRegistry`，直接调用自定义渲染函数。
