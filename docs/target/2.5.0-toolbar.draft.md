# 工具栏

> 悬浮 pill 形状操作栏，提供项目操作、执行控制和视图切换。默认居中固定在窗口顶部，可拖动移位，叠加在画布之上。（变更：原为固定顶部操作栏，改为悬浮 pill 可拖动）

## 操作分组

### 项目操作

| 操作 | 说明 |
|------|------|
| 新建 | 创建空白项目 |
| 打开 | 从文件系统打开项目 |
| 保存 | 保存当前项目（Cmd+S） |

文件导出通过图中的 SaveImage 等节点完成，不在工具栏。

### 执行控制

| 操作 | 说明 |
|------|------|
| 运行 | 手动触发执行 |
| 取消 | 中断当前执行 |

运行中"运行"按钮变为"取消"。

### 视图切换

| 操作 | 说明 |
|------|------|
| 面板显隐 | 切换预览面板、AI 对话面板的显示 |
| 布局切换 | 切换预设布局（单屏 / 双屏） |
| 亮/暗模式 | 切换主题模式 |

## API

工具栏项通过注册机制声明，不硬编码。新增功能只需注册一个 `ToolbarItem`：

```rust
toolbar.register(ToolbarItem {
    id:      "run",
    section: Section::Execution,
    icon:    Icon::Play,
    label:   "运行",
    action:  Action::Run,
    enabled: Box::new(|| !engine.is_running()),
    visible: Box::new(|| true),
});
```

`enabled` 和 `visible` 是运行时求值的闭包，驱动按钮的禁用/隐藏状态。同一 `Section` 内的项按注册顺序排列，Section 之间用分隔线隔开。

## 外观与位置

- **形状**：pill（胶囊形），高度约 40px，圆角全圆
- **默认位置**：窗口顶部居中，距顶边 12px
- **可拖动**：拖拽 pill 本体移动至任意位置
- **层级**：浮于画布之上（FloatingPanel 容器）

## 交互

所有工具栏操作交给交互控制器处理，工具栏自身不调用引擎 API。
