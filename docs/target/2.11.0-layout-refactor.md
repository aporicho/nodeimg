# 布局引擎重构：消除中间 LayoutBox 转换

> 状态：待实施

## 问题

当前布局流程有一层多余的转换：

```
PanelNode → 构造 LayoutBox 临时树 → Flexbox 计算 → 写回 PanelNode
```

行业标准做法（Flutter、SwiftUI、Qt）是 Flexbox 引擎直接操作树节点，不需要中间临时树。

## 方案

定义 `LayoutTree` trait，Flexbox 引擎通过 trait 操作任意树。

### LayoutTree trait

```rust
// widget/layout/types.rs

pub trait LayoutTree {
    type NodeId: Copy;

    /// 读取节点的布局样式
    fn style(&self, node: Self::NodeId) -> &BoxStyle;

    /// 获取子节点列表
    fn children(&self, node: Self::NodeId) -> Vec<Self::NodeId>;

    /// 直接把算出的位置写入节点
    fn set_rect(&mut self, node: Self::NodeId, rect: Rect);
}
```

三个方法 = Flexbox 引擎需要的全部能力。

### Flexbox 引擎改为泛型

```rust
// widget/layout/mod.rs
fn layout<T: LayoutTree>(tree: &mut T, root: T::NodeId, available: Rect);

// widget/layout/measure.rs
fn measure<T: LayoutTree>(tree: &T, node: T::NodeId) -> DesiredSize;

// widget/layout/arrange.rs
fn arrange<T: LayoutTree>(tree: &mut T, node: T::NodeId, available: Rect, ...);
```

### PanelTree 实现 LayoutTree

```rust
// panel/tree/layout.rs

struct PanelLayoutAdapter<'a> {
    tree: &'a mut PanelTree,
    renderer: &'a mut Renderer,  // Widget::layout() 需要 Renderer 做文字度量
}

impl LayoutTree for PanelLayoutAdapter<'_> {
    type NodeId = NodeId;

    fn style(&self, node: NodeId) -> &BoxStyle { /* 读 PanelNode 的样式 */ }
    fn children(&self, node: NodeId) -> Vec<NodeId> { /* 读 PanelNode 的子节点 */ }
    fn set_rect(&mut self, node: NodeId, rect: Rect) { /* 直接写 PanelNode.rect */ }
}
```

`PanelLayoutAdapter` 把 `&mut PanelTree` 和 `&mut Renderer` 包在一起，是 Rust 借用规则要求的语法包装，不是逻辑上的额外层。

### Widget trait 变化

```rust
// 之前
fn layout(&self, renderer: &mut Renderer) -> LayoutBox;

// 之后
fn layout(&self, renderer: &mut Renderer) -> BoxStyle;
```

Widget 只声明"我要多大"，不再构造 LayoutBox 中间节点。

### 删除的类型

- `LayoutBox` — 中间临时树节点，不再需要
- `LayoutResult` — 中间结果容器，不再需要（结果直接写入树节点）

## 涉及文件

| 文件 | 操作 |
|------|------|
| `widget/layout/types.rs` | 新增 LayoutTree trait，删除 LayoutBox / LayoutResult |
| `widget/layout/measure.rs` | 参数改为泛型 `<T: LayoutTree>` |
| `widget/layout/arrange.rs` | 参数改为泛型，直接调 set_rect |
| `widget/layout/mod.rs` | layout() 签名改为泛型 |
| `widget/trait.rs` | Widget::layout() 返回 BoxStyle 而非 LayoutBox |
| `widget/atoms/button.rs` | layout() 返回 BoxStyle |
| `panel/tree/layout.rs` | 改为 PanelLayoutAdapter 实现 |
| `demo.rs` | layout 调用方式变化 |
