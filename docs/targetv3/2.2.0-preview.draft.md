# 预览面板

> 显示选中节点的输出预览。自建 Canvas widget，GPU 纹理直接在 wgpu 层渲染，无 CPU 回读。

## 功能

- 随画布选中节点联动，显示该节点的输出图像
- 支持缩放（滚轮）和平移（拖拽）查看
- 支持像素级检查：鼠标悬停时显示当前像素坐标和 RGBA 值
- 选中节点无图像输出时显示空白占位

## 渲染机制

预览面板是自建的 `Shader` widget，在 `Primitive::prepare()` / `draw()` 回调里直接操作 wgpu render pass。引擎产出的 GPU 纹理（`GpuTexture`）留在显存，面板直接 sample 并输出到屏幕，不经过 CPU。

```
引擎 GpuTexture（显存）
    ↓ sample（wgpu render pass）
预览面板输出（屏幕）
```

iced 只提供 widget 容器和鼠标/键盘事件投递，渲染完全由面板自己控制。

## 显示内容

| 内容 | 说明 |
|------|------|
| 图像 | 选中节点的输出图像，按面板尺寸适应缩放 |
| 直方图 | 叠加显示 RGB 直方图（可选开关） |
| 像素信息 | 悬停坐标 + RGBA 值，显示在面板角落 |
| 缩放比例 | 当前缩放倍率，显示在面板角落 |

## 刷新时机

交互控制器在以下情况通知预览面板刷新：
- 画布选中节点变化
- 引擎执行完成（节点输出更新）
