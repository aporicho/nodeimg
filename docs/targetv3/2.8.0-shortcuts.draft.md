# 快捷键系统

> 全局和上下文快捷键管理，将按键组合映射为操作消息，发送给交互控制器或画布控制器。

## 快捷键表

### 全局（任意焦点区域有效）

| 快捷键 | 操作 |
|--------|------|
| Cmd+S | 保存项目 |
| Cmd+Z | 撤销 |
| Cmd+Shift+Z | 重做 |
| Cmd+N | 新建项目 |
| Cmd+O | 打开项目 |
| Space+拖拽 | 平移画布 |

### 画布焦点

| 快捷键 | 操作 |
|--------|------|
| Cmd+C | 复制选中节点 |
| Cmd+V | 粘贴节点 |
| Cmd+X | 剪切选中节点 |
| Cmd+A | 全选节点 |
| Delete / Backspace | 删除选中节点 |
| Cmd+D | 复制并偏移（Duplicate） |
| F | 聚焦到选中节点 |
| Cmd+Shift+A | 打开节点库搜索 |

### 预览面板焦点

| 快捷键 | 操作 |
|--------|------|
| F | 适应窗口大小 |
| 1 | 100% 缩放 |

## 机制

- **上下文路由**：快捷键系统按当前焦点区域路由消息——画布快捷键发给画布控制器，全局快捷键发给交互控制器。
- **冲突处理**：上下文快捷键优先于全局快捷键。焦点在画布时，Delete 删除节点而非触发其他全局操作。
