# 节点库面板

> 所有可用节点的浏览和搜索入口，支持常驻侧边面板和右键弹出菜单两种形态。

## 形态

| 形态 | 触发方式 | 说明 |
|------|----------|------|
| 常驻侧边面板 | 工具栏显隐开关 | 固定在界面左侧，始终可见 |
| 右键弹出菜单 | 画布空白处右键 | 在鼠标位置弹出，选择后关闭，添加的节点放置在右键位置 |

两种形态共享同一套节点列表和搜索逻辑，仅展示容器不同。

## 内容结构

节点按分类分组展示，分类可折叠：

```
▼ 输入 / 输出
    加载图像
    保存图像
▼ 颜色调整
    亮度 / 对比度
    色相 / 饱和度
    曲线
▶ 滤镜
▶ AI 生成
...
```

## 搜索

- 关键词实时过滤，匹配节点名称
- 有搜索词时隐藏分类分组，平铺显示匹配结果
- 搜索框在两种形态下均可用；Cmd+Shift+A 在画布焦点下直接聚焦搜索框

## 添加节点

- **侧边面板**：双击节点名，或拖拽到画布指定位置
- **右键菜单**：单击节点名，节点添加到右键触发位置

添加操作产生消息交给交互控制器，由交互控制器调用引擎 `add_node` API。
