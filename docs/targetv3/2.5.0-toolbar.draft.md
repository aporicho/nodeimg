# 工具栏

> 顶部操作栏，提供项目操作、执行控制和视图切换。

## 操作分组

### 项目操作

| 操作 | 说明 |
|------|------|
| 新建 | 创建空白项目 |
| 打开 | 从文件系统打开项目 |
| 保存 | 保存当前项目（Cmd+S） |
| 导出 | 导出当前选中节点的输出结果 |

### 执行控制

| 操作 | 说明 |
|------|------|
| 运行 | 手动触发执行（mode=manual） |
| 取消 | 中断当前执行 |

运行中"运行"按钮变为"取消"。

### 视图切换

| 操作 | 说明 |
|------|------|
| 面板显隐 | 切换节点库、预览面板、AI 对话面板的显示 |
| 布局切换 | 切换预设布局（如单屏 / 双屏） |

## 交互

所有工具栏操作产生消息后交给交互控制器处理，工具栏自身不调用引擎 API。
